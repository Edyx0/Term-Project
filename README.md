# Online Shopping Simulation

## Project Overview
A terminal-based e-commerce platform that handles product browsing, cart management, and checkout processing.

## Features
- Customer Flow: Browse catalog, add to cart, and generate receipts.
- Admin Flow: Manage inventory and view sales analytics.
- Data Persistence: All data is saved to JSON files between sessions.

## File List
- main.py: Entry point and menu system.
- catalog.py: Handles product loading and searching.
- cart.py: Logic for adding and removing items from the cart.
- orders.py: Processes checkouts and updates stock.
- storage.py: Basic file read and write functions.
- tests.py: Automated tests for system validation.

## Setup
1. Place all Python files in the same directory.
2. Run the command: python main.py
3. Follow the on-screen menu prompts.
