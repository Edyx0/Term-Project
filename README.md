ONLINE SHOPPING SIMULATION

Project Overview
A terminal-based e-commerce platform for products and checkout.

Features
- Customer Flow: Catalog, cart, and receipts.
- Admin Flow: Dashboard and inventory.
- Data Persistence: JSON storage.

File List
- main.py: Menu system.
- catalog.py: Products loading.
- cart.py: Cart logic.
- orders.py: Checkout processing.
- storage.py: File helpers.

Setup
1. Place files in same directory.
2. Run: python main.py
3. Follow prompts.
